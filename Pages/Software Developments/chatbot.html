This Javascript chatbot leverages Rivescript, a Perl based scripting language for chatbots. This project contains the following components:
  • brain.rive

    • Contains the text-based rivescript file that gives the chatbot personality.

  • chatbot.js

    • Makes the chatbot alive by controlling the chatbot's interactions/actions on the webpage.

  • chatbot.html

    • Allows for the chatbot to be deployed onto the webpage.

  • chatbotStyle.css

    • Controls the chatbot's appearance on the website.

  • riveFileGenerator.py

    • Generates the text-based rivescript file based on data from excel sheet.

What can it do?
  • Tell it hello or hi
  • Say: My name is ... and it will greet you!
  • Say: I like the color ...
  • Tell it how you feel like... I'm super tired
  • Or, even ask what it could do ... what can you do?
  • Make it google somthing... google pineapples
  • Thank the bot!
  • Tell it to say something random
  • Ask it philosophical questions like... what are you
  • What is up
  • You are a bot
  • Mom told me to say hello
  • Tell it how old you are.. I am 15 years old
  • What is your cell number
  • I am wearing a red shirt
